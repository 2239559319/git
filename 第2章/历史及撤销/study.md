### 查看提交历史

 - git log:按照时间序列输出仓库中的所有提交，包括
    - SHA-1校验码
    - 作者姓名和邮箱
    - 提交日期
    - 提交信息
 - git log -p: 显示每次提交所引入的差异
    - git log -p -2 只输出最近两次提交
 - git log --stat :输出每次提交的简要信息
    - 改动的文件列表
    - 共有多少文件被改动
    - 文件有多少新增行和删除行
  
### 撤销操作

 - git reset HEAD <file>把文件移除暂存区
 - git checkout -- [file]  (未提交的文件)撤销对文件的修改
